# 🎓 Student Management System in C++

A simple console-based Student Management System built using C++. This project is designed for beginners to understand and implement basic programming concepts and CRUD operations (Create, Read, Update, Delete).

---

## 📌 Features

- Add new student records
- Display all student records
- Search student by roll number
- Update student information
- Delete specific student record
- Delete all records
- Menu-driven interface

---

## 🧠 Concepts Used

- Arrays
- Functions
- Loops (for, while)
- Conditional statements (if-else, switch-case)
- Strings
- Basic logic building

---

## 🗂️ Student Data Fields

Each student record contains:

- Name  
- Roll Number  
- Course  
- Section  
- Contact Number  

---

## ▶️ How to Run

### Compile the program:
```bash
g++ main.cpp -o sms
````

### Run the program:

```bash
./sms
```

### On Windows (MinGW):

```bash
g++ main.cpp -o sms.exe
sms.exe
```

---

## 📋 Menu Options

1. Enter student data
2. Show all students
3. Search student
4. Update student
5. Delete student record
6. Exit

---

## ⚠️ Limitations

* Uses fixed-size arrays (max 50 students)
* No file handling (data is not saved permanently)
* Console-based interface only
